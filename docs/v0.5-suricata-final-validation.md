# v0.5 Suricata / Network Telemetry Final Validation

## Scope

This validation confirms the end-to-end Suricata network telemetry integration for Sovereign AI SOC v0.5.

Validated capabilities:

- Suricata Docker sensor runtime
- Suricata EVE JSON ingest worker
- `network_events` persistence
- read-only Network Events API
- Incident Network Evidence API
- Network Activity frontend page
- Incident Detail Network Evidence panel
- Health observability
- AI Incident Brief enrichment
- Incident report enrichment
- Case report enrichment
- Analyst Evidence Pack enrichment

## Validation principles

The integration remains:

- read-only
- local-first
- human-in-the-loop
- evidence enrichment only
- not an IPS
- not SOAR automation
- not automatic incident creation


Expected frontend behavior:

- Network Activity page loads Suricata telemetry.
- Incident Detail keeps the existing UX and layout.
- Network Evidence appears as read-only context below the Structured context / Audit trail area.
- Health shows Suricata sensor, Suricata ingest and Latest Network event freshness.

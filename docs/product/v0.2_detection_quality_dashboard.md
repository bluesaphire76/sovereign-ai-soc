# v0.2 — Detection Quality Dashboard

## Purpose

The Detection Quality Dashboard measures how defensive synthetic scenarios are observed by the Sovereign AI SOC pipeline.

It is designed to support detection engineering and lab validation across:

- synthetic event generation;
- Wazuh ingestion;
- AI SOC incident creation;
- incident correlation;
- recommended priority assignment;
- MITRE signal visibility;
- investigation workflow validation.

## Current Scope

This first implementation is frontend-only and consumes the existing `/incidents` API.

It searches for synthetic incidents using the `SYNTHETIC` keyword and calculates:

- number of synthetic incidents visible in AI SOC;
- percentage of correlated synthetic incidents;
- percentage of HIGH or CRITICAL synthetic incidents;
- percentage of synthetic incidents with MITRE signal;
- per-scenario breakdown;
- latest synthetic incidents.

- [ ] Can see all users.
- [ ] Can create users.
- [ ] Can modify users.
- [ ] Can enable/disable users.
- [ ] Can delete users.
- [ ] Can reset other users' passwords.
- [ ] Can perform privileged SOC actions.

- [ ] Cannot see Security Audit.
- [ ] Can access Users only in self-service mode.
- [ ] Can reset own password.
- [ ] Cannot create users.
- [ ] Cannot modify roles.
- [ ] Cannot enable/disable/delete users.
- [ ] Can operate on incidents/cases according to policy.

### VIEWER

- [ ] Cannot see Security Audit.
- [ ] Can access Users only in self-service mode.

- [ ] Cannot modify roles.

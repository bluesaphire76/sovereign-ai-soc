# v0.2 — Synthetic Attack Scenarios

## Purpose

Synthetic Attack Scenarios provide controlled, defensive, lab-only security events that can be collected by Wazuh and used to validate Sovereign AI SOC end-to-end.

The goal is to test:

- Wazuh ingestion;
- AI incident analysis;
- correlation engine;
- case grouping;
- case AI analysis;
- executive dashboard;
- report export.

These scenarios do not execute real attacks, exploitation, scanning, credential guessing, malware, or unauthorized activity.

## Design Principle

The implementation generates synthetic JSON log events and writes them to a dedicated local log file.

## Release Boundaries

v0.6.0 does not claim:

- autonomous remediation;
- arbitrary SOAR execution;
- external system remediation;
- shell/SSH command execution;
- firewall, identity, endpoint or host mutation;
- production-ready external playbook orchestration.

Controlled SOAR remains limited to allowlisted internal product workflow records.
